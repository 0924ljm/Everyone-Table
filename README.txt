-참고 사이트: 우리의식탁 

2.페이지 구성

===공통,로그인===
기능1: 메뉴바 마우스 호버시 언더바가 해당 매뉴의 위치와 가로사이즈에 맞게 변함

기능2: 로그인 클릭시 로그인 팝업창 열기( ID:1234@naver.com PW:1234)	
        	-아이디,비밀번호 입력여부확인
	-아이디에@ 포함되었는지 형식확인
	-아이디 형식 맞을시 아이디 존재여부 확인
	-아이디 확인시 비밀번호 일치여부 확인
	-5회이상 잘못 입력시 경고
	-아이디/비밀버호 전부 제대로 입력시 로그인 버튼이 회원 아이디로 바뀜
	-회원 아이디 클릭시 회원정보창 보여줌(로그아웃 버튼 포함)


===메인페이지===
         
기능1: 추천레시피 섹션 - 버튼 클릭시 슬라이드기능
         
기능2: 베스트레시피 섹션 - swiper기능 활용 호버시 효과
		          각아이템을 itemlist파일에서 정보를 받아서 보여줌
		          아이템 클릭시 해당아이템 정보를 넘겨받는 "레시피 페이지"로 이동

기능3: 레시피 섹션 - 카테고리 클릭시 해당 카테고리 아이템 정보를 itemlist파일에서 받아서 보여줌
		  더보기 클릭시 해당 카테고리의 아이템 정보를 추가로 보여줌
		  위와 마찬가지로 아이템 클릭시 해당아이템 정보를 넘겨받는 "레시피 페이지"로 이동

기능4: 스크롤 위치에 따른 각섹션 에니매이션 효과



===레시피 페이지===
기능1: 페이지를 열기전 클릭한 아이템의 정보를 받아서 보여줌

기능2: 레시피 재료 정보를받아오고 재료정보 하단에있는 구매하러가기 버튼 클릭시 
        해당재료정보(재료이름,가격)를 카트페이지로 넘겨줌



===카트 페이지===
기능1: 레시피페이지 에서 넘겨받은 재료 정보를 보여줌

기능2: 플러스/마이너스 버튼 클릭시 아이템 수량 더하고 빼서 수량에맞는 가격정보를 보여줌 

기능3: 각각의 체크박스 클릭시 체크된 아이템의 가격은 최종가격정보에 합산되고 체크되지 않은 아이템의 가격은 제외함
         전체선택 클릭시 모든 체크박스를 컨트롤함
